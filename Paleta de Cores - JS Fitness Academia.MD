# Guia de Identidade Visual e Paleta de Cores

Este documento detalha a paleta de cores extraída da imagem de referência e sugere variações tonais para suporte ao desenvolvimento da interface do aplicativo.

---

## 1. Núcleo de Profundidade (Escuros)

As cores de fundo estabelecem o contraste e a hierarquia visual primária.

* **Cores Identificadas:** Preto (`#1a1a1a`) e Cinza Escuro (`#333333`).
* **Explicação:** Ideal para superfícies de fundo (*backgrounds*), barras de navegação e estados de elevação em temas escuros.

### Variações Sugeridas
| Tom | Código Hex | Aplicação Recomendada |
| :--- | :--- | :--- |
| **Sombra Profunda** | `#0d0d0d` | Para áreas de foco total. |
| **Cinza Médio** | `#4d4d4d` | Para divisores de seção ou bordas. |
| **Cinza Suave** | `#737373` | Para textos secundários ou ícones desativados. |

---

## 2. Núcleo de Atenção (Destaques)

Tons quentes e vibrantes que guiam o olhar do usuário.

* **Cores Identificadas:** Amarelo Ouro (`#fdb913`) e Amarelo Vibrante (`#ffcc00`).
* **Explicação:** Devem ser usadas em elementos de *Call to Action* (CTA), botões primários e notificações importantes.

### Variações Sugeridas
| Tom | Código Hex | Aplicação Recomendada |
| :--- | :--- | :--- |
| **Âmbar Intenso** | `#e6a700` | Para estados de *hover* ou botões pressionados. |
| **Amarelo Pastel** | `#ffe680` | Para fundos de alertas ou *badges* sutis. |
| **Ouro Queimado** | `#cc9900` | Para detalhes que exigem um toque mais sóbrio. |

---

## 3. Núcleo de Claridade (Metálicos/Neutros)

Tons neutros para garantir legibilidade e sofisticação.

* **Cores Identificadas:** Prata (`#e0e0e0`) e Branco (`#ffffff`).
* **Explicação:** Essenciais para textos principais, logotipos e ícones de alta visibilidade sobre fundos escuros.

### Variações Sugeridas
| Tom | Código Hex | Aplicação Recomendada |
| :--- | :--- | :--- |
| **Branco Neve** | `#f2f2f2` | Para textos longos, reduzindo a fadiga ocular em telas OLED. |
| **Prata Acinzentado** | `#b3b3b3` | Para legendas e metadados. |
| **Platina** | `#d1d1d1` | Para estados de foco em campos de entrada (*input*). |
